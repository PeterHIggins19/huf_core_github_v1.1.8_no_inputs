How to Run
01
Clone the repository

Get the HUF core repo. All cases are in cases/ with their own README and data files.

git clone https://github.com/PeterHiggins19/
  huf_core_github_v1.1.8_no_inputs
cd huf_core_github_v1.1.8_no_inputs
02
Run a bundled case

Each case has a run.py with synthetic data built in. No downloads required.

python cases/planck/run.py
python cases/markham/run.py
python cases/traffic/run.py
03
Check outputs

Each run produces a JSONL provenance log and prints per-step C(ℋ), α*, items_to_cover_90pct, and any detected anomalies.

planck_huf_run.jsonl
markham_huf_audit.jsonl
traffic_huf_run.jsonl
04
Use real data

For Planck: install healpy and set --use-real-data. For Markham: export CSV from markham.ca capital plan.

python cases/planck/run.py \
  --use-real-data \
  --map LFI_SkyMap_070_2048.fits
05
Partner adapters

VDB partner adapters live in adapters/. Each exports JSONL from the VDB SDK and feeds into HUF normalization.

python adapters/weaviate/export.py \
  --host localhost:8080
python adapters/qdrant/export.py
06
Stability checks

Each run includes a stability packet: C(ℋ) > threshold, Var(ρ) bound, items_to_cover_90pct, and JSONL hash for tamper-evident logging.

# Stability checks printed at end:
# ✅ C(ℋ) = 0.968 > 0.95
# ✅ Var(ρ) = 0.0003 < 0.01
# ✅ items_90pct = 7 >= 5
