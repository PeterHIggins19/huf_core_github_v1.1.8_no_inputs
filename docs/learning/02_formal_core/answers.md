# Answers — Formal core (conservative)

1. A finite-dimensional space V with a nonnegative weight assignment ρ that sums to 1.
2. Transformations should preserve the relevant normalized totals (global and/or local sums).
3. A sub-collection normalized locally (its weights sum to 1 within the regime).
4. No equivalence claims to advanced mathematical frameworks or cross-domain theorems.
5. Because public credibility depends on reproducible, bounded claims that can be audited.
6. AS (shares) and EB (discard accounting), supported by CM (rankings) and TR (decisions).
